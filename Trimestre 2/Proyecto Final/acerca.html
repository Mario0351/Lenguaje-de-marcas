Acerca de este Proyecto

Descripción General

Este proyecto es un Dashboard informativo que integra datos de clima y noticias. Ha sido desarrollado como proyecto final para el módulo de "Lenguaje de Marcas y Sistemas de gestión de la información".

Objetivos del Desarrollo

  • Aplicar estructuras HTML semánticas y diseño responsive con CSS (usando Bootstrap).
  • Manipular el DOM y gestionar eventos con JavaScript.
  • Integrar información dinámica desde fuentes de datos (en este caso, archivos JSON locales simulando APIs).
  • Organizar el proyecto en múltiples páginas HTML interconectadas.
  • Documentar el proceso de desarrollo y presentar el proyecto de manera clara.

Fuentes de Datos (JSON Locales)

Para este proyecto, se han utilizado archivos JSON locales para simular la respuesta de APIs públicas de clima y noticias. Esto permite trabajar con datos dinámicos sin necesidad de keys de API o de conexión a internet para la demostración.

  • Datos de Clima: `clima.js` (simulando WeatherAPI).
  • Datos de Noticias: `noticias.js` (simulando GNews API).

Tecnologías Empleadas

  • HTML: Para la estructura semántica de las páginas web.
  • CSS: Para el diseño y la responsividad, utilizando en gran medida la librería Bootstrap 5.
  • JavaScript: Para la interactividad, la manipulación del DOM y la carga/procesamiento de los datos JSON.
  • Bootstrap: Framework CSS para un diseño rápido y responsivo.
